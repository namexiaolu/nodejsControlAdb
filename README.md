# nodejsControlAdb
用nodejs实现同时控制多台安卓设备
准备工作


安装node
在nodejs官网直接下载安装，会自动嫁娶环境变量。nodejs.cn/download
安装node-cmd
npm install --save npm-cmd
或者全局安装
npm install --save npm-cmd
(如果安装比较慢可以替换淘宝的npm源：> npm config set registry registry.npm.taobao.org)

安装adb并设置环境变量
